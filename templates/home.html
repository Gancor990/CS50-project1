This is a tool you can use to search for books within a database by using the title, author or ISBN number as keywords. You can then leave reviews on books of your choice.

This is a showcase of Python using Flask, Jinja2 and SQLAlchmey as well as the Goodreads API.

Thanks to Harvard's CS50 Web Programming with Python & JavaScript course, Corey Schafer's excellent YouTube Flask tutorials as well as the Philip Robinson Library @ Newcastle University for being a great place to hole up and code :)
